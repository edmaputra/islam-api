Assalamualaikum Warahmatullahi Wabarokatuh

Selamat Datang di Islam API by edmaputra. API ini dibuat untuk para rekan-rekan yang ingin membuat suatu aplikasi yang di dalamnya ada fitur untuk menampilkan waktu shalat untuk daerah tertentu di seluruh dunia.

Terima Kasih untuk PrayTimes.org untuk metode perhitungan waktu shalat, dan Google Maps API untuk mendukung pencarian lokasi.

Parameter Query yang bisa digunakan adalah sebagai berikut :

Berdasarkan Tanggal (default value : tanggal hari ini)

URL :
shalat?d=ddMMyyyy
  • Parameter 'd' untuk tanggal
  • Format Tanggal ddMMyyyy
Contoh:<
  • https://islam.edmaputra.com/shalat?d=08072018

Berdasarkan Lokasi (default value : "samarinda")

URL :
shalat?l=lokasi
  • Parameter 'l' untuk lokasi
  • Jika ada spasi dalam pencarian lokasi, ganti dengan karakter "+"
  • Case-Insensitive
Contoh:
  • https://islam.edmaputra.com/shalat?l=samarinda
  • https://islam.edmaputra.com/shalat?l=Jalan+Kolonel+Soetadji
  • https://islam.edmaputra.com/shalat?l=tenggarong+kutai+kartanegara

Berdasarkan Zona Waktu (Timezone) (default value : 8.0)

URL :
shalat?t=timezone
  • Parameter 'l' untuk lokasi
  • Rentang Timezone -10 s/d 14
Contoh:
  • https://islam.edmaputra.com/shalat?t=-4

Berdasarkan Metode Perhitungan (default value : 3)

URL :
shalat?m=metode
  • Parameter 'm' untuk Metode Perhitungan
  • Rentang Metode 0 s/d 6
    • 0 : Ithna Ashari
    • 1 : University of Islamic Sciences, Karachi
    • 2 : Islamic Society of North America (ISNA)
    • 3 : Muslim World League (MWL)
    • 4 : Umm al-Qura, Makkah
    • 5 : Egyptian General Authority of Survey
    • 6 : Institute of Geophysics, University of Tehran
Contoh:
  • https://islam.edmaputra.com/shalat?m=5

Semua Parameter di atas bisa dikombinasikan tanpa memperhatikan urutan parameternya

Contoh :
  • https://islam.edmaputra.com/shalat?l=samarinda&d=08072019
  • https://islam.edmaputra.com/shalat?d=11112011&l=balikpapan+selatan&m=5
  • https://islam.edmaputra.com/shalat?m=5&t=9&l=new+york
